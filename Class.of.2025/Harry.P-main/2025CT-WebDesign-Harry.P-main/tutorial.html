Printing "Hello World" in Lua

The following Lua code prints "Hello World" to the console:

print("Hello World")

Explanation of the Code

print Function:
The print function in Lua is used to output text or other data to the standard output, typically the console or terminal. It is a built-in function in Lua, which means you do not need to import or require any libraries to use it.

Parentheses ():
The parentheses are used to pass arguments to the print function. In this case, the argument is the string "Hello World".

String "Hello World":
The text "Hello World" is a string literal in Lua. A string literal is a sequence of characters enclosed in double quotes ("). In Lua, you can also use single quotes ('), but double quotes are more common for strings that include spaces.

Step-by-Step Execution

When the Lua interpreter executes the code print("Hello World"), it calls the print function. The print function takes the string "Hello World" as its argument. The function then outputs the string to the console or terminal, displaying the text Hello World to the user.

You should see the output Hello World displayed in the terminal.
